# Changelog

All notable changes to Chronicle Keeper will be documented in this file.

## [1.0.0] - 2026-02-07

### Added
- Initial release
- Ollama integration for local LLM hosting
- Multi-layered memory system (short-term, long-term, entity-based)
- Vector embeddings for semantic memory search
- Chat slash commands (/ai, /narrate, /npc, /memory, /dm)
- Combat narration and dice roll interpretation
- D&D 5e system integration
- Character sheet data access
- Scene and journal awareness
- Memory browser for viewing and managing memories
- Configurable system prompts
- Connection health monitoring
